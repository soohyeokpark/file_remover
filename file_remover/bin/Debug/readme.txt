*-*-*-*-*-*-FILE REMOVER 메뉴얼*-*-*-*-*-*-

타겟 경로 = 관리 하고자 하는 최상위 폴더의 경로 (그 하위에 있는 것 관리 함)

완전삭제 유무 ex) 예 = 휴지통을 거치지 않고, 영원히 삭제 합니다

보존일  ex) -1 = 오늘과, 오늘 -1 일 까지는 데이터를 보존합니다
          ex) -0 = 오늘의 데이터만 보존 합니다 (과거~ 어제꺼 다 지움)

수행기간 = 매일 hh시 mm분에 동작 할지 정합니다 (군대 날짜식으로)

테스트 = 테스트 모드일 시, 시간과 관계없이 동작을 1회 진행 합니다

잠금 및 적용 = 변경된 설정 내용을 config 파일에 적습니다, 
                   컨트롤을 잠그어 사용자의 실수를 방지 합니다

타겟 드라이브 여유 = 타겟 경로가 위치한 모 드라이브의 여유공간을 표시
